The CSS Box Model

We can declare widths and heights on elements to define their size. We also have other properties like margins, padding, and borders which make up the box model.

Margin

Margin is used to create space around an element, outside of its border.

Padding

Padding is used to create space around an element's content, inside of its border.

Borders

Borders are used to create a visible line around an element's content, inside of its padding.
